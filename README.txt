== Description ==
This plugin allows store owners to create a fully customizable add-to-cart drawer, providing a seamless and user-friendly shopping experience. The drawer appears when a customer adds a product to their cart, displaying a summary of the items in the cart, along with the option to edit quantities, remove items, or proceed to checkout.
